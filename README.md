# Use

- CommandLine/Terminal
```
git clone https://github.com/yuki-funasaka-uhuru/todoapp.git
cd /your_folder_path/todoapp
npm start
```

- Browse
  - http://localhost:3000/
  - http://localhost:3000/signup/
  - http://localhost:3000/signin/
